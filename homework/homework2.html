Unordered List

  • < body >: Similar to the html tag, there can only be one body tag per document. It is used to indicated everything that is within the html document such as paragraphs, images, headings, lists and everything else.
  • < br >: This tag has no end tag and is used to indicate a line break on the page.
  • < p >: Used to indicated a paragraph, browsers will add a single blank line before and after each p tag.
  • < a href >: is used to add a hyper link to another webpage from your webpage. The href part of this element is used to indicated the intended destination for a link.

Ordered List

  1. < ul >: Will add an unordered list, using bullet points to dictate each item.
  2. < ol >: Will add an ordered list using either numerals or the alphabet to indicate items on the list.
  3. < li >: This tag and it's end tag are required before and after every item on each list, either ordered or unordered, in order to let the code know that it is, in fact, an item on the list.
  4. < img >: This tag is used to embed images onto HTML pages. According to W3 Schools, the img tag has two required attributes:
    src = Defines the path to the images
    alt = Adds alternate text for the image for screen readers or if the image cannot be displayed for some reason
  5. < strong >: Used to accentuate important text that will usually be bolded.
  6. nbsp: The nonbreaking space entity in html.
